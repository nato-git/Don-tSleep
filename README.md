# 睡眠監視プログラム

## 内容

このプログラムは主にOpenCV、YOLOを用いた睡眠防止プログラムです。  
PCのカメラから映っている顔を取得し、その耳の傾きから計算しています。  
寝ていると判断された場合、その時刻が記録された写真を撮影され、PCの背景画像がその画像に変更されます。  
また、寝ている際に音が流れるので自身でフォルダ内にお好きな音楽 sound.mp3 を入れてください。

## 注意点

寝ていると判断されると背景画像が強制的に変更されます。元の背景画像はWriteImageに保存されますが、背景が変更されたまま二回目のプログラム実行をすると画像が上書きされてしまうので注意してください。また、サウンドが流れます。音量に注意してください。

## インストール必要なライブラリ

```cmd
pip install --upgrade pip

pip install opencv-python 

pip install ultralytics

pip install playsound3
```

## 実行方法

このフォルダをダウンロードし、自身の環境に上記のライブラリをインストールしてください。  
インストールが完了し、sound.mp3をフォルダ内に入れたらこのプログラムを実行してください。
